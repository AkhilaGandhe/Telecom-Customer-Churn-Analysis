# Telecom Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn behaviour in a telecom company using Power BI. The dashboard helps identify key churn drivers, customer demographics, contract patterns, and service-related issues that impact customer retention.

## Tools & Technologies

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

## Business Problem

Customer churn directly impacts revenue and customer lifetime value. This analysis aims to identify:

- Why customers are leaving
- Which customer segments are at risk
- The impact of contracts on churn
- The relationship between service calls and churn

## Key Insights

- Overall Churn Rate is 26.86%.
- Competitor offers are the leading reason for customer churn.
- Customers citing "Competitor had better offers" represent the largest churn segment.
- Month-to-Month contract customers contribute the highest churn.
- Higher service call frequency is associated with increased churn.
- Older customer groups show higher churn tendencies.
